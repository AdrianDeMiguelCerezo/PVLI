# THE SOUTH BORDER
Este proyecto es un trabajo universitario

## Descripción del juego
The South Border es un juego roguelike RPG por turnos y de aventuras de texto. Ambientado en el Viejo Oeste, juegas como un forajido tratando de escapar cruzando la frontera de América mientras está en busca y captura.
